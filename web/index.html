This service allows you to generate an ICS file for your school's menu on Linq Connectc using a Siri shortcut or API call.

How to Use

  1. Install the Siri shortcut
  2. OPTIONAL: Add the ICS to Calendar shortcut
  3. Run the shortcut and provide the following information:
    • Building ID
    • District ID
    • Start Date (MM-DD-YYYY)
    • End Date (MM-DD-YYYY)
  4. The shortcut will generate and download the ICS file for you

More information on how to get the building and district IDs can be found here.

API Endpoint

For advanced users, you can directly use our API endpoint:

POST https://schoolmenuconnector.com/get-menu

Note: Only POST requests are supported for this endpoint.

Parameters (send as form data):

  • buildingId
  • districtId
  • startDate
  • endDate
