School Menu Connector

This service allows you to generate an ICS file for your school's menu using a Siri shortcut.

How to Use

  1. Install the Siri shortcut (link coming soon)
  2. Run the shortcut and provide the following information:
    • Building ID
    • District ID
    • Start Date (MM-DD-YYYY)
    • End Date (MM-DD-YYYY)
  3. The shortcut will generate and download the ICS file for you

API Endpoint

For advanced users, you can directly use our API endpoint:

POST https://schoolmenuconnector.com/get-menu

Parameters (send as form data):

  • buildingId
  • districtId
  • startDate
  • endDate
